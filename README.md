# Tropical Precipitation Response to Anthropogenic Climate Change in Recent Decades

Codes used to plot the figures of 'Tropical Precipitation Response to Anthropogenic Climate Change in Recent Decades' submitted to Nature Communications.



This repository contains the figure-generation and analysis codes used in the study:



“Tropical precipitation response to anthropogenic climate change.”

(submitted to Nature Communications)



The scripts are designed to reproduce the key figures and diagnostics presented in the manuscript.
